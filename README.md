# Проект Банковских карт

## Описание:

Виджет, который показывает несколько последних успешных банковских операций клиента. 

## Установка:

1. Клонируйте репозиторий:
```
https://github.com/AndyTums/MyProject.git
```
2. Установите зависимости: 
```
poetry install
```
